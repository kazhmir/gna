# Many Chat Rooms

To run the server:

    go run . -server :8888

To run the client:

    go run . -client :8888 -name glenda

To send messages with the client, just type stuff and press enter.

The client commands are:

    /room <int>
To change room (Instance)

    /num
To know how many people are in the room.

    /rename <string>
To rename

    /quit
To quit
